# ModuleBuild
Module build to apk demo 
利用组件（Module）开发apk实例
#### 包含的组件有
**include ':app', ':baselibrary', ':tasks:ordermanager', ':tasks:pointmanager', ':orderlibrary'**
**':baselibrary'是公共组件**
其中 "app"、":tasks:ordermanager"为application组件
其余的":baselibrary"、":tasks:pointmanager"、":orderlibrary"为lib组件
